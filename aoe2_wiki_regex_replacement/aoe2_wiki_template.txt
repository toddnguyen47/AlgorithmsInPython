== Overview ==
This template's purpose is to fill in any ''Age of Empires II'' unit, technology, building, or civilization linked and prefixed with their respective icon in paragraphs or lists. The user doesn't have to look up the icon name or if there is a page of the same name in another game of the series.
It looks like this: {{AoEII Icons|Champion}}.

== How to use it ==
While the template itself is rather large, it is very simple to use and has only one mandatory input parameter, the name of the unit, technology, building, or civilization.

It knows every unit/technology/building/civilization, as well as the plurals.

Simply call the template and fill in the full name as it appears in the game.

'''Syntax:'''

<code>{{t|AoEII Icons|Name|Image size|linek break}}</code>

The template has 3 parameters: 
# The name of the unit/technology/building. For accepted inputs see known inputs below
# The desired image size. Intended for it is a number. Default is 22.
# String between icon and name. To add a line break type in ''<nowiki><br></nowiki>''. Default is a [[wikipedia:Non-breaking space|non-breaking space]].

Only the first parameter input is mandatory for the template to work, the latter two parameters are intended for use in tables. When unused, the default values are in effect, which is standard for use in lists.


=== Examples ===
#1
<code><nowiki>{{AoEII Icons|Spearman}}</nowiki></code>

gives

{{AoEII Icons|Spearman}}

#2
<code><nowiki>{{AoEII Icons|Spearman|45|<br>}}</nowiki></code>

gives

{{AoEII Icons|Spearman|45|<br>}}

#3
<code><nowiki>{{AoEII Icons|Archers}} are ranged units. They are trained at the {{AoEII Icons|Archery Range}} and are strong against {{AoEII Icons|Long Swordsmen}}.</nowiki></code>

gives

{{AoEII Icons|Archers}} are ranged units. They are trained at the {{AoEII Icons|Archery Range}} and are strong against {{AoEII Icons|Long Swordsmen}}.


#4

<code><nowiki> Cavalry armor upgrades are:</nowiki>

<nowiki>*{{AoEII Icons|Scale Barding Armor}}</nowiki>

<nowiki>*{{AoEII Icons|Chain Barding Armor}}</nowiki>

<nowiki>*{{AoEII Icons|Plate Barding Armor}}</nowiki></code>

gives

Cavalry armor upgrades are:

*{{AoEII Icons|Scale Barding Armor}}

*{{AoEII Icons|Chain Barding Armor}}

*{{AoEII Icons|Plate Barding Armor}}


The template can also be called by using the redirect 2icons:

<code>{{t|2icons|Name}}</code>
